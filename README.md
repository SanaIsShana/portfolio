## Getting Started                  
### Prerequiesties
- Node.js >=16.0.0
- npm
  
### Installation
1. Clone
```bash
https://github.com/SanaIsShana/portfolio.git
```
2. Install Dependencies
```bash
npm install
```
3. Run the code
```bash
npm run dev
```
